# Task 5 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, missing values, and relationships using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Titanic Dataset (train.csv)

## Analysis Performed
- Dataset Information using `.info()`
- Statistical Summary using `.describe()`
- Missing Value Analysis
- Value Counts using `.value_counts()`
- Histogram
- Boxplot
- Scatter Plot
- Pair Plot
- Correlation Heatmap

## Key Findings
- The dataset contains 891 passenger records with 12 features.
- Missing values were found in the Age, Cabin, and Embarked columns.
- Most passengers were between 20–40 years of age.
- The majority of passengers traveled in Third Class.
- More passengers did not survive than survived.
- Ticket fares contained several outliers.
- Correlation analysis showed mostly weak relationships among numerical variables.

## Outcome
Successfully performed exploratory data analysis using statistical summaries and visualizations to understand the dataset and identify meaningful insights before further data preprocessing or predictive modeling.
